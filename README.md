# wrds (EDA / research repo)

This repository is **intentionally not production code**.

- It contains exploratory analysis, iteration, experiments, and notebooks.
- Code quality, structure, and reproducibility may vary over time.
- Large data extracts and caches are **intentionally not committed** (GitHub has a 100MB per-file limit).

## What’s excluded from git

See `.gitignore` for the authoritative list. In particular, large CSV extracts (multi‑GB) and `final_project/pkl_cache/` are ignored.
